자바스크립트를 다시 공부하던 중 컴파일러 강의에서 자주 듣던 용어들이 보여서 새로 정리한다.

# 스코프
자바스크립트는 함수 레벨 스코프(Function level scope)를 사용한다. 일반적으로 사용하는 블록 레벨 스코프(Block level scope)와는 다르다. 

예시
```javascript
function f() {
    if (true) {
        var x = 0;
    }
    console.log(x);     //0 출력
}
```
다음과 같이 함수 레벨 스코프는 말 그대로 스코프가 블록이 아니라 함수이다.
자바스크립트에서 블록 레벨 스코프를 이용하기 위해서는 var 대신 let이나 const를 사용하면 된다.

또 자바스크립트는 렉시컬 스코프(lexical scope)를 가진다. 렉시컬 스코프란 스코핑에 있어서 함수의 호출 위치가 아닌 선언 위치를 확인하는 것이다.

예시
```javascript
var text = 'global';

function foo() {
	console.log(text);
}

function bar() {
	var text = 'bar';
	foo();
}

bar();  //global 출력
```
위와 같이 실행 위치가 아니라 선언 위치에서 스코핑을 한다.

# 호이스팅
자바스크립트에서의 호이스팅은 선언 명령을 함수의 시작부분으로 끌어올리는 것을 말한다. 이때 값을 대입하는 명령은 원래 위치에 그대로 둔다.

# 클로저
자바스크립트에서 클로저(Closure)는 외부 함수의 실행이 종료되더라도 내부 함수가 외부 함수에 접근할 수 있게 하는 것이다. 간단히 말하자면 외부 함수가 종료되도 외부 함수의 변수를 가지고 내부 함수가 실행될 수 있는 것이다.

```javascript
function foo() {
    var a = [];
    var i;
	
    for(i = 0; i < 3; i++){
      a[i] = function() {
        return i;
        }
    }
    return a;
  }
  
  var b = foo();

  console.log( b[0]() );    //3 출력
  console.log( b[1]() );    //3 출력
  console.log( b[2]() );    //3 출력
```
